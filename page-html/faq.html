Une question ?

Le délai de livraison est de 3 à 5 jours ouvrés. Nous vous envoyons un mail de confirmation lorsque votre commande est expédiée. Si vous n’avez pas reçu votre commande dans les délais indiqués, contactez-nous.

Nous acceptons les paiements par carte bancaire (Visa, Mastercard, American Express) et par Paypal. Nous n’acceptons pas les chèques bancaires, les virements ou les paiements à la réception. Les paiements sont sécurisés et vos données bancaires ne sont pas conservées. Si vous rencontrez un problème de paiement, contactez-nous.

Les frais de livraison sont de 5€ pour toute commande inférieure à 50€. La livraison est offerte pour toute commande supérieure à 50€. Les frais de livraison sont indiqués sur la page de paiement, après avoir renseigné votre adresse de livraison. Si vous rencontrez un problème de livraison, contactez-nous.

Vous pouvez retourner un article dans un délai de 14 jours après réception de votre commande. Les articles doivent être retournés dans leur emballage d’origine, neufs et non portés. Les frais de retour sont à votre charge. Si vous souhaitez retourner un article, contactez-nous.

Vous pouvez suivre votre commande sur la page de suivi de commande. Vous recevrez également un mail de confirmation lorsque votre commande est expédiée. Si vous n’avez pas reçu votre commande dans les délais indiqués, contactez-nous.
